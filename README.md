# MPC 电机控制项目

这是一个基于模型预测控制（MPC）的电机控制项目，用于永磁同步电机（PMSM）的高性能控制。

## 项目结构

```
MPC/
├── app/                    # 应用程序入口和工具
├── core/                   # 核心算法模块
│   ├── control/           # 控制算法（MPC）
│   ├── identify/          # 电机参数辨识
│   └── model/             # 电机数学模型
├── hal/                   # 硬件抽象层
├── modules/               # 功能模块
│   ├── compensation/      # 逆变器补偿
│   ├── fallback/          # PI后备控制
│   ├── observer/          # 观测器
│   ├── outer_loop/        # 外环控制
│   ├── safety/            # 安全保护
│   └── sensing/           # 传感器处理
├── tools/                 # 分析工具
├── ui/                    # 用户界面
└── docs/                  # 文档
```

## 主要特性

1. **模型预测控制（MPC）**：基于离散时间模型的电流环控制
2. **电机参数辨识**：在线电参数辨识算法
3. **多环控制**：位置环、速度环、电流环级联控制
4. **安全保护**：过流、过压、失速保护
5. **观测器**：扰动观测器、负载转矩观测器
6. **补偿模块**：逆变器死区补偿、器件压降补偿

## 编译和运行

```bash
# 编译控制演示
make

# 运行控制演示
./mpc_demo

# 编译辨识演示
make identify_demo

# 运行辨识演示
./identify_demo

# 清理构建文件
make clean
```

## 配置选项

在 `app/control_main.c` 中可以配置：
- 控制模式（位置/速度）
- FRF频域扫频测试
- 电机标幺化参数
- 保护阈值
- 观测器参数
